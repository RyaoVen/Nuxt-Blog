# RyBolg
RyBlog是一项基于Nuxt.js和SpringBoot的博客系统，具有完整的后台管理功能。

## 技术栈

* 前端：Nuxt.js + Element-ui + axios　＋ TypeScript + Pinia
* 后端：SpringBoot + MyBatis-Plus + MySQL + Redis + Nginx + Docker
* 测试：FoxApi + JUnit5

## 功能

### 前台

1. 登录
2. 注册
3. 文章列表
4. 文章详情
5. 个人页
6. 留言页
7. 个人动态页
